git --version             - вывод версии Git
git init                  - инициализация папки проэкта для Git
git status                - на какой ветке сейчас находимся или проверить вносились ли изменения 
git add git.txt           - добавление файла git.txt для отслеживания
git commit -m "my commit" - сохранение изменений 
git add .                 - добавление изменений всех файлов в директории после нужно сохранить изменения сомандою commit
